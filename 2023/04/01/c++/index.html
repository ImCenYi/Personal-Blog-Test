引用的概念

某个变量的引用，==等价于==这个变量，相当于这个变量的==别名==

引用(&)语法如下

int n=4;
int &r=n;

对于n的运算，r同样也会做相同的运算。
反过来，对于r的运算，同样也会作用到n上。

如

int n=7;
int &r=n;
r=4;     //这一步运算也作用到了n上
cout<<r;
cout<<n;
n=5;     //这一步运算也作用到了r上
cout<<r;

最后r=n=5，可见引用运算同步。

注意，引用是从一而终的，不可以引用其他变量
引用只能引用==变量==，不能引用常量和表达式

这里r1=b这一句，并不是让r1引用b，而是给r1赋了b的值
